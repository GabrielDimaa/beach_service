## :clipboard: Descrição do Projeto
<p align="justify">Beach Service é um software para plataforma Android e IOS, canal de compra e venda que vai facilitar o contato entre vendedores ambulantes e consumidores. No software teremos dois tipos de usuários: consumidores e vendedores. O consumidor poderá solicitar os produtos que deseja pelo aplicativo, e assim o vendedor irá até o local solicitado.
Quando o vendedor se cadastrar no aplicativo ele poderá se registrar escolhendo o determinado tipo de produto que vende, para quando o comprador solicitar um item estar relacionado com seu tipo de venda.
No contexto da solicitação do usuário consumidor, após isso a aplicação vai dar match com vendedores de acordo com o pedido. O vendedor poderá aceitar o pedido do consumidor, logo depois, calcular o tempo da entrega de acordo com a distância entre ambos.
</p>
